Explore.js is an open-source project that helps designers, artists, teachers, and amateur astronomers visualize the cosmos. It is composed of a JavaScript library that unfies numerous existing libraries and provides useful tools to provide a unified view of space. It also contains a data set with textures, scripts to access current data from multiple sources, and positional data.

If it's your first time here, checkout the demo section to see what the library can do.
