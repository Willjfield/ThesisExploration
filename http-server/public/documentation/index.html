      Planets

      xpl.SolarSystem(Planet,Time)
      This function returns the position of a planet at a specified time. The planet objects can be accessed from the xpl.planets planets array. The time is entered as a Julian Day. The returned position is described in an XYZ coordinate system with the Z-axis as up and the Sun's barrycenter at the origin. For a demonstration of this function, look at Drawing Planets in P5 //Look through each planet in the Solar System //and find its current position for(var p=0;p< xpl.planets.length;p++){ var planet = xpl.planets[p] var position = xpl.SolarSystem(planet,xpl.now) }

      xpl.planets(Planet,Time)
      xpl.planets is an array of Planet Javascript objects. Planet objects contain all of the data needed to derive the position, rotation and obliqueness of each planet in our Solar System given a certain time. //Look through each planet in the Solar System //and store it as a variable called planet for(var p=0;p< xpl.planets.length;p++){ var planet = xpl.planets[p] }

      Time

      xpl.now
      xpl.now holds the current time as a Julian Date as of when the script is loaded. Julian dates (abbreviated JD or Jday) are the number of days since noon Universal Time on January 1, 4713 BCE (on the Julian calendar). For a detailed explanation of Julian Dates and an online converter tool, look at The US Naval Obsrvatory's Julian Date Converter //Find planet Earth and determine its current position //as of when the script is executed var planet = xpl.planets[2] var position = xpl.SolarSystem(planet,xpl.now) console.log(position)

      xpl.updateTime()
