Explore.js is an open-source project that helps designers, artists, teachers, and amateur astronomers visualize the cosmos. It is composed of a JavaScript library that unfies numerous existing libraries and provides useful tools to provide a unified view of space. It also contains a data set with textures, scripts to access current data from multiple sources, and positional data. Explore.js began as Will Field's Master's thesis at the Interactive Telecommunications Program at New York University.
