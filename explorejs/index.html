Explore.js is an open source project that helps designers, artists, teachers and amateur astronomers visualize the cosmos using JavaScript. Explore.js is an open-source project whose main objective is to make it easier to create interactive visualizations of astronomy. Explore.js is composed of a javascript library that unfies numerous existing libraries and provides useful tools. It also contains a data set with textures, scripts and positional data. Explore.js began as Will Field's masters thesis at the Interactive Telecommunications Program at New York University.
